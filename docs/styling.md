# Tailwind CSS Configuration

This project uses Tailwind CSS via Nuxt UI for styling. The configuration is automatically handled by the `@nuxt/ui` module.

## Custom Styles

Additional custom styles can be added to `assets/css/main.css`.

## Dark Mode

Dark mode is automatically supported through Nuxt UI and can be toggled using the header button.

## Icons

Icons are provided by Heroicons through Nuxt Icon. You can use icons with the format:
```vue
<UIcon name="i-heroicons-home" />
```
